Welcome to jaql.org!

Jaql is a new query language being developed for JSON data. This site is still under construction, but you can read about and try Jaql using the following links.

New Release!
Release 0.3: Overview, gzipped tar file
Jaql's current release adds the following to the 0.2 release:

  • New syntax: please see the overview.
  • More robust query rewrites.
  • Bug fixes.
  • Multiple versions of Hadoop supported: Jaql + Hadoop 0.16.*, Jaql + Hadoop 0.15.3 and Jaql + Hadoop 0.17.1
To learn more about jaql, read the overview and get started by downloading. Feedback is greatly appreciated!
